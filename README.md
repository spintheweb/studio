# Spin the Web Studio

Spin the Web Studio is a software for managing Spin the Web sites. In this context, sites may be web sites, portals, intranet, extranets, web apps or web services.

The difference between a regular site and a Spin the Web site is the presence of a WBDL file. The WBDL (Webbase Description Language) file describes the site: its contents, routing, authorization and internationalization.

## Software as a Service (SaaS)
Head over to https://studio.spintheweb.org, visit a Spin the Web site, authenticate and start developing.
